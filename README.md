# COVID-19 Data Analysis Dashboard

## 📌 Project Overview
An end-to-end Data Analytics project analyzing COVID-19 cases, deaths, recovery trends and regional impact using Python, SQL and Power BI.

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- MySQL
- Power BI
- Data Cleaning
- Data Visualization

## 📊 Project Workflow

### 1. Data Cleaning & EDA (Python)
- Cleaned COVID-19 dataset
- Removed inconsistencies
- Handled missing values
- Performed exploratory data analysis

### 2. SQL Analysis
Performed SQL queries to analyze:
- Total confirmed cases
- Total deaths
- Recovery rate
- Country-wise impact
- Region-wise COVID trends

### 3. Power BI Dashboard
Created an interactive dashboard including:
- Total Cases KPI
- Total Deaths KPI
- Recovery Rate KPI
- Active Cases KPI
- Top affected countries
- WHO Region analysis
- Global COVID map visualization
